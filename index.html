🚀 Configuração Inicial (Nexus ESECCinfãesCTE)

Cria um Google Sheets novo e abre Extensões → Apps Script
Cola o código do ficheiro Nexus_ESECCinfaesCTE_Code.gs e implementa como Aplicação Web
Abre o Professor, cola o URL do Apps Script em ⚙️ Gerir → Google Sheets
Configura turma, alunos, locais de estágio e sessões de aula
Clica 📡 Publicar Configuração — os alunos recebem tudo automaticamente
Distribui o link do Aluno e o URL do Sheets aos alunos (uma única vez)
