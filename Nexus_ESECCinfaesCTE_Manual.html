Manual de Utilização

Nexus ESECCinfãesCTE — Sistema de avaliação, autoavaliação e controlo de presenças

📦 1. Criar o Repositório GitHub

Pré-requisitos

Conta GitHub gratuita · Conta Google · Os 4 ficheiros do sistema

  1. Abre github.com e faz login (ou cria conta)
  2. Clica no + (canto superior direito) → New repository
  3. Nome: nexus-eseccinfaescte · Visibilidade: Public · Marca "Add a README"
  4. Clica Create repository
  5. No repositório, clica Add file → Upload files
  6. Arrasta os 4 ficheiros: index.html, ..._Professor.html, ..._Aluno.html, ..._Code.gs
  7. Escreve uma mensagem de commit e clica Commit changes

Ativar o GitHub Pages

  1. Vai a Settings (separador no topo do repositório)
  2. No menu lateral, clica Pages
  3. Source: Deploy from a branch · Branch: main · Pasta: / (root)
  4. Clica Save · Aguarda 1–2 minutos
  5. O URL aparece: https://teuuser.github.io/nexus-eseccinfaescte/
ℹ️
O repositório deve ser público para o GitHub Pages gratuito funcionar. Não há risco de segurança — os dados estão no Google Sheets (privado), não no repositório.

📊 2. Configurar o Google Sheets

  1. Abre sheets.google.com e cria um ficheiro novo
  2. Dá um nome descritivo: Nexus — 3.ºB EAC — 2025/2026
  3. Vai a Extensões → Apps Script
  4. Apaga o código existente no editor
  5. Abre o ficheiro Nexus_ESECCinfaesCTE_Code.gs num editor de texto, copia tudo e cola
  6. Clica no ícone 💾 para guardar

Implementar como Aplicação Web

  1. No Apps Script, clica Implementar → Nova implementação
  2. Tipo: Aplicação Web
  3. Executar como: "Eu"
  4. Quem tem acesso: "Qualquer pessoa"
  5. Clica Implementar e autoriza quando pedido
  6. Copia o URL gerado — começa por https://script.google.com/macros/s/...
⚠️
Se o Google mostrar "Esta app não foi verificada", clica Avançado → Ir para (nome do projeto). É normal para scripts pessoais.

🔗 3. Ligar ao Sistema

  1. Abre o ficheiro Professor (via GitHub Pages ou localmente)
  2. Vai a ⚙️ Gerir → Google Sheets
  3. Cola o URL do Apps Script e clica 🔌 Testar
  4. Deve aparecer ✅ Ligação estabelecida

📡 Publicar Configuração

O botão 📡 Publicar Configuração (topo do painel Gerir) envia toda a configuração para a folha "Configuração" do Google Sheets. O ficheiro do Aluno lê automaticamente esta configuração ao abrir.

🔄
Sempre que alterares a configuração (novos alunos, nova sessão de aula, novo local), clica 📡 Publicar para os alunos receberem as alterações.

🗳️ Avaliação de Pares

Configuração (⚙️ Gerir → Configuração de Pares)

Define a sessão (turma, curso, tipo), os grupos e membros, e os critérios de avaliação (até 10, com emoji, descrição, escala 1–5★). Os critérios são convertidos para 0–20 valores automaticamente.

Fluxo do Aluno

O aluno seleciona o grupo que está a apresentar → avalia com estrelas em cada critério → opcionalmente avalia cada membro individualmente → submete. Cada dispositivo só pode votar uma vez por grupo.

Resultados (📈 Resultados)

Ranking por média, comparação entre grupos, comentários e avaliação individual por membro. Importação de votos do Sheets para ver resultados em qualquer dispositivo.

✍️ Autoavaliação

Define vetores (ex: Assiduidade, Participação, Trabalho), cada um com peso percentual e até 4 características. Os alunos avaliam-se de 1–5★ em cada característica. A nota final é ponderada pelos pesos.

Suporta múltiplas disciplinas/UFCDs — cada uma cria um tab separado no Sheets (ex: AA_EAC, AA_UFCD6025).

📍 Presenças de Estágio

Configuração (⚙️ Gerir → 📍 Presenças de Estágio)

SeparadorO que configurar
📊 PainelVista diária em tempo real de todos os alunos
👥 AlunosAté 20 alunos, cada um com código de 4 dígitos e local atribuído
📍 LocaisCoordenadas GPS, raio, horário de entrada/saída por local
📄 RelatóriosRelatório individual filtrável, impressão, CSV

Horário e Pontualidade

Define horário global (ex: 09:00–17:30) com tolerâncias configuráveis. Cada local pode ter horário próprio. Ao marcar entrada:

AtrasoEstado
≤ 10 min✅ Presente
10–15 min⚠️ Falta de pontualidade
> 15 min❌ Falta

GPS

O sistema verifica se o aluno está dentro do raio do local atribuído. Cada registo inclui coordenadas, distância ao centro, e indicação "No estágio?" (Sim/Não).

Pausas

O aluno pode registar pausas (🍽️ Almoço, ☕ Lanche, 🚶 Intervalo, ou motivo personalizado) sem ter de fazer saída + nova entrada. As pausas têm cronómetro em tempo real e são subtraídas automaticamente das horas contabilizadas.

Feedback Diário

Após marcar entrada, o aluno pode opcionalmente descrever o que fez e avaliar o dia de 0 a 10.

📝 Aulas Práticas

Configuração (⚙️ Gerir → 📝 Aulas Práticas)

SeparadorO que configurar
📅 SessõesDisciplina, módulo, sumário, data, hora início, duração, tolerâncias
👥 AlunosAté 20 alunos, cada um com código + palavra-passe individual
📊 PainelVista em tempo real da sessão ativa
📄 RelatóriosRelatório por aluno/disciplina, impressão, CSV

Criar sessão de aula

Preenche disciplina, módulo (opcional), sumário (visível ao aluno), data, hora início, duração e tolerâncias. Clica 📅 Criar e Abrir Sessão. Só pode haver uma sessão ativa de cada vez.

Fluxo do Aluno

Introduz código + palavra-passe → vê a sessão ativa com sumário → clica ✋ Marcar Presença → recebe estado automático (Presente / Pontualidade / Falta) → pode registar pausas → pode escrever feedback e avaliar a aula de 0 a 10.

Barras de progresso

O painel do professor mostra barras de horas acumuladas por aluno, com contagem de presenças, pontualidade e faltas.

📄 Relatórios

Cada secção (Estágio e Aulas) tem um separador de relatórios que permite:

  • Filtrar por aluno e disciplina/período
  • Gerar relatório detalhado com tabela dia-a-dia
  • 🖨️ Imprimir — abre janela de impressão formatada
  • 📊 CSV — exporta para Excel/Calc

Os relatórios incluem: data, horários de entrada/saída/marcação, atrasos, estados, pausas, avaliação diária e feedback do aluno.

🔄 Nova Turma / Ano Letivo

No fim do ano, em ⚙️ Gerir → Nova Turma:

AçãoO que faz
💾 Exportar TemplateGuarda ficheiro JSON com locais, critérios, horários (sem alunos/dados)
📂 Importar TemplateCarrega a estrutura de um template anterior
🔄 Nova Turma (mesmo Sheets)Apaga alunos e dados, mantém estrutura e Sheets (dados ficam como arquivo)
🆕 Nova Turma + Novo SheetsApaga tudo incluindo URL — terás de criar Sheets novo
📥
Ambos os botões de "Nova Turma" exportam um backup completo automático antes de apagar.

🔑 Acesso do Aluno

Na primeira utilização, o aluno abre o link do Aluno.html e cola o URL do Google Sheets (fornecido pelo professor). O URL fica guardado no telemóvel com validade de um dia — à meia-noite expira e pede novo login nas secções que requerem código.

A configuração (grupos, sessões, locais) é carregada automaticamente do Sheets a cada abertura.

🗳️ Votar — Avaliação de Pares

  1. Seleciona o grupo que está a apresentar
  2. Avalia cada critério com 1–5 estrelas
  3. Opcionalmente avalia os membros individualmente
  4. Opcionalmente deixa um comentário
  5. Clica ✓ Submeter Avaliação
🔒
A avaliação é totalmente anónima — nenhum nome é associado às notas. Cada dispositivo só pode avaliar cada grupo uma vez.

✍️ Autoavaliação

  1. Introduz o primeiro e último nome
  2. Avalia cada característica de cada vetor com 1–5 estrelas
  3. A nota final é calculada automaticamente com os pesos definidos pelo professor
  4. Clica ✅ Submeter

📍 Presenças de Estágio

  1. Introduz o código de acesso (4 dígitos, fornecido pelo professor)
  2. Vê o estado GPS (dentro/fora do raio do local de estágio)
  3. Clica ▶ Marcar Entrada — o sistema regista coordenadas e calcula pontualidade
  4. Durante o dia, pode registar pausas (almoço, lanche, etc.) com cronómetro
  5. Pode escrever o que fez hoje e avaliar o dia de 0 a 10
  6. No fim, clica ⏹ Marcar Saída
📡
O GPS deve estar ativo no telemóvel. A precisão varia com o dispositivo e ambiente (melhor ao ar livre). O raio de tolerância é configurável pelo professor.

📝 Presenças em Aulas

  1. Introduz código + palavra-passe
  2. Vê a aula ativa (disciplina, módulo, sumário, horário)
  3. Clica ✋ Marcar Presença
  4. Recebe estado automático: ✅ Presente / ⚠️ Pontualidade / ❌ Falta
  5. Pode registar pausas e escrever feedback

⏰ Horários de Estágio

O sistema verifica a pontualidade dos alunos ao marcar entrada no estágio, com base num horário configurável.

Horário global (⚙️ Gerir → 📍 Presenças → 👥 Alunos)

Na card "Configuração de Horas e Horário", define:

CampoDefaultDescrição
Hora de entrada09:00Hora esperada de chegada
Hora de saída17:30Hora esperada de saída
Tolerância pontualidade10 minAté este atraso = Presente
Tolerância falta15 minAcima disto = Falta

Horário por local

Cada local de estágio pode ter horário próprio (em 📍 Locais). Se definido, sobrepõe o horário global para os alunos desse local. Útil quando diferentes empresas têm horários distintos.

Cálculo automático

Quando o aluno marca entrada:

AtrasoEstado
≤ tolerância pontualidade✅ Presente
Entre pontualidade e falta⚠️ Falta de pontualidade
> tolerância falta❌ Falta

O estado e minutos de atraso ficam registados no Google Sheets (colunas "Atraso" e "Pontualidade") com formatação colorida automática.

📚 Multi-Autoavaliação

O sistema suporta múltiplas autoavaliações simultâneas na mesma turma — por exemplo, UFCD 6025, UFCD 6026 e EAC ao mesmo tempo.

Configuração (Professor)

  1. Vai a ⚙️ Gerir → ✍️ Autoavaliação
  2. Configura a primeira disciplina (nome, vetores, pesos)
  3. Clica 💾 Guardar Atual na Lista
  4. Clica ＋ Adicionar Disciplina e repete para cada disciplina
  5. Clica 📡 Publicar para enviar todas para os alunos

Fluxo do Aluno

Se houver mais de uma disciplina, o aluno vê um ecrã com cards — uma por disciplina. Clica numa, preenche, submete. Após submeter, vê um ecrã de sucesso com três opções:

  • 🔄 Repetir — corrigir erros ou outro aluno preencher no mesmo dispositivo
  • 📚 Outra disciplina — volta à seleção
  • 🏠 Início — volta ao ecrã principal

As disciplinas já submetidas ficam marcadas com ✅.

🏫 Gestão de Turmas

O sistema suporta múltiplas turmas em simultâneo no mesmo browser, cada uma com os seus dados e Google Sheets independentes.

Criar turma

  1. Clica no ＋ ao lado do seletor de turma (barra superior)
  2. Preenche o nome (ex: "3.ºB EAC"), ano letivo, e URL do Sheets
  3. Clica ➕ Criar Turma

Alternar entre turmas

Seleciona a turma no dropdown da barra superior. A página recarrega com os dados dessa turma.

Importar backup

No modal de turmas, clica 📂 Importar Backup para criar uma turma a partir de um ficheiro JSON de backup anterior.

Recuperar após limpar cookies

  1. Cria a turma com o URL do Sheets que tinhas
  2. Vai a ⚙️ Gerir → 📥 Importar do Sheets
  3. Toda a configuração publicada é recuperada

📵 Modo Offline

Quando o dispositivo perde ligação à internet, aparece um banner vermelho no topo:

📵
Sem ligação à internet — os dados ficam guardados e serão enviados quando a rede voltar.

O que funciona offline:

  • A interface continua funcional (todos os dados estão em localStorage)
  • Votos e presenças são guardados localmente
  • Quando a rede volta, os votos em fila são enviados automaticamente

O que não funciona offline: publicar configuração, importar do Sheets, submeter autoavaliações.

📲 QR por Sessão de Aula

Cada sessão de aula aberta tem um botão 📲 QR que gera um QR code projetável com:

  • Link direto para o Aluno.html
  • Nome da disciplina e módulo
  • Horário da sessão
  • Sumário (se definido)

O professor projeta o QR na sala → os alunos leem com o telemóvel → abrem o sistema → introduzem código + senha → marcam presença.
